First steps

One diagram is defined by a JSON array. The array contains the title of the diagram and a list of components, which will be presented in it as boxes.

Here is an empty diagram:

Input:

["A Graphic"]

Output:

["A Graphic"]

A box is defined again by an array which must contain at least one element, the name of the box, but can contain a lot more.

Input:

["A Graphic", ["A Box"]]

Output:

["A Graphic", ["A Box"]]

Setting Box Position

The position of each box is a two-character string, which uses the convention used in the chess board (except that I don't play chess so the numbers are from top to bottom). For example "A1" positions the block at the top left corner, "A2" is on the right of "A1" etc.

The size of the board is calculated automatically and it can go up to 9.

Input:

["Box Position", 
 ["A1", "A1"],
 ["A2", "A2"],
 ["A3", "A3"],

 ["B1", "B1"],
 ["B2", "B2"],
 ["B3", "B3"],
 ["C1", "C1"],
 ["C2", "C2"],
 ["C3", "C3"]
]

Output:

["Box Position", ["A1", "A1"], ["A2", "A2"], ["A3", "A3"], ["B1", "B1"], ["B2", "B2"], ["B3", "B3"], ["C1", "C1"], ["C2", "C2"], ["C3", "C3"] ]

Children

Components are composed of smaller components. In Graphs this is reflected by the ability for each box to have children which are shown only when the user clicks on it.

Input:

["Children", 
 ["A Parent", "a2", 
   ["Child 1", "a2"],
   ["Child 2", "b2"],
   ["Child 3", "c2"]
 ]

]

Output:

["Children", ["A Parent", "a2", ["Child 1", "a2"], ["Child 2", "b2"], ["Child 3", "c2"] ] ]

You can have more than two levels. Each level has different CSS attributes

Input:

["Children", 
  ["First", 
    ["Second",
      ["Third"]
    ]
  ]
] 


Applying Styling

  
Stress on a certain box by applying different CSS styling.

  
To add styling, include a property with one of the CSS styles which are defined as graphs constants: 'important', 'minor'

  
  
Input:

  
["Box Position", 
 ["A1", "A1"],
 ["A2", "A2"],
 ["A3", "A3"],

 ["B1", "B1"],
 ["B2", "B2"],
 ["B3", "B3"],
 ["C1", "C1"],
 ["C2", "C2"],
 ["C3", "C3"]
]

 
Output:

 
["Styling",
 ["Ordinary Box1", "A1"],
 ["Ordinary Box2", "A2"],
 ["A very important box", "A3"],

 ["Ordinary Box3", "B1"],
 ["Ordinary Box4", "B2"],
 ["Not Very Important Box", "B3"],

]

Output:

["Children", ["First", ["Second", ["Third"] ] ] ]
