First steps

One diagram is defined by a JSON array. The array contains the title of the diagram and a list of components, which will be presented in it as boxes.

Here is an empty diagram:

Input:

["A Graphic"]

Output:

["A Graphic"]

A box is defined again by an array which must contain at least one element, the name of the box, but can contain a lot more.

Input:

["A Graphic", ["A Box"]]

Output:

["A Graphic", ["A Box"]]

Setting Box Position

The position of each box is a two-character string, which uses the convention used in the chess board (except that I don't play chess so the numbers are from top to bottom). For example "A1" positions the block at the top left corner, "A2" is on the right of "A1" etc.

The size of the board is calculated automatically and it can go up to 9.

Input:

["Box Position", 
 ["A1", "A1"],
 ["A2", "A2"],
 ["A3", "A3"],

 ["B1", "B1"],
 ["B2", "B2"],
 ["B3", "B3"],
 ["C1", "C1"],
 ["C2", "C2"],
 ["C3", "C3"]
]

Output:

["Box Position", ["A1", "A1"], ["A2", "A2"], ["A3", "A3"], ["B1", "B1"], ["B2", "B2"], ["B3", "B3"], ["C1", "C1"], ["C2", "C2"], ["C3", "C3"] ]

Setting Box Size

The size of the board is calculated automatically and it can go up to 9.

Input:

["Box Size", 
 ["2X2", "A1", "2x2"],
 ["1X1", "C1"],
 ["1X2", "C2", "1X2"],
 ["2X1", "A3", "2X1"]
]

Output:

["Box Size", ["2X2", "A1", "2x2"], ["1X1", "C1"], ["1X2", "C2", "1X2"], ["2X1", "A3", "2X1"] ]

Adding Children

Components are composed of smaller components. In Graphs this is reflected by the ability for each box to have children which are shown only when the user clicks on it.

Input:

["Children", 
 ["A Parent", "a1",
   ["Child 1", "a2"],
   ["Child 2", "b2"],
   ["Child 3", "c2"]
 ]

]

Output:

["Children", ["A Parent", "a2", ["Child 1", "a2"], ["Child 2", "b2"], ["Child 3", "c2"] ] ]

You can have more than two levels. In each level has its own CSS styles

Input:

["Children", 
  ["First", 
    ["Second",
      ["Third"]
    ]
  ]
]

Output:

["Children", ["First", ["Second", ["Third"] ] ] ]

Applying CSS Styles

Put a stress on a certain box by applying different CSS style.

To change the box styling, include a one of the semantic styling properties which are defined as constants in Graph: important and minor

Input:

["Styling",
 ["Ordinary Box1", "A1"],
 ["A very important box", "A2", "important"],
 ["Ordinary Box2", "A3"],
 ["Ordinary Box3", "B1"],
 ["Ordinary Box4", "B2"],
 ["Not So Important Box", "B3", "minor"]
]

Output:

["Styling", ["Ordinary Box1", "A1"], ["A very important box", "A2", "important"], ["Ordinary Box2", "A3"], ["Ordinary Box3", "B1"], ["Ordinary Box4", "B2"], ["Not So Important Box", "B3", "minor"] ]

One semantic styling property corresponds two several CSS styles, one for each of the levels of the diagram:

Input:

["Level-Aware Styling",
 ["Ordinary Box from Level One", "A1","2X1"],
 ["Important Box from Level One", "A2", "important","2X1",
  ["Ordinary Box from Level Two", "A1","2X1"],
  ["Important Box from Level Two", "A2", "important","2X1"]
 ]
]

Output:

["Level-Aware Styling", ["Ordinary Box from Level One", "A1","2X1"], ["Important Box from Level One", "A2", "important","2X1", ["Ordinary Box from Level Two", "A1","2X1"], ["Important Box from Level Two", "A2", "important","2X1"] ] ]

You can add a styling property of your own: Declare it in the 'css_styles' array and define CSS styles for it.

Applying Special Styles

Put a stress on a certain box by applying different CSS style.

One semantic styling property corresponds two several CSS styles, one for each of the levels of the diagram:

Input:

["Transparency",
 ["Normal Box"],
 ["Transparent Box", "A2", "transparent",
  ["You can...", "A1"],
  ["...see this", "b1"]
 ]
]

Output:

["Transparency", ["Round Box", "circle"], ["Transparent Box", "A2", "transparent", ["You can...", "A1"], ["...see this", "b1"] ] ]
